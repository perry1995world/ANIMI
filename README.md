# ANIMI 웹 프로토타입

ANIMI는 사진 속 사람을 자연스러운 실제 동물로 치환하는 AI 카메라 서비스입니다.

## 호스팅

https://perry1995world.github.io/ANIMI/

## 파일럿 v0.1

1. 앱 유형: 웹 기반 카메라 프로토타입
2. 핵심 기능: 사진 속 모든 사람을 실제 고양이로 치환
3. 고양이 스타일: 옷/액세서리 없는 자연 털 고양이
4. 털색 기준: 상의 컬러 80% + 머리색 20%
5. 털색 제한: 실제 고양이에게 존재하는 자연 털색 범위
6. 배경 처리: 원본 배경, 조명, 카메라 각도, 구도 유지
7. 결과 생성: 촬영 1회당 1장
8. 현재 상태: AI API 연결 전 UI/UX 프로토타입

## 실행

정적 파일이라 바로 열 수 있지만, 카메라 권한은 `localhost`에서 가장 안정적으로 동작합니다.

```bash
python3 -m http.server 5173
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:5173
```

## AI 연결 지점

`app.js`의 `transformImage()` 내부가 실제 이미지 변환 API 연결 지점입니다.

전달해야 할 값:

1. 원본 이미지: `selectedImage`
2. 파일럿 프롬프트: `pilotPrompt`

현재는 변환 API 대신 결과 화면 UX 확인용 캔버스 이미지를 생성합니다.
